# Big L Driving School Website
This is a professional landing page project for **Big L Driving School**.

## Features Added:
* **Interactive Service Cards**: Cards that lift and turn red on hover.
* **Navigation Bar**: A clean menu for Home, Lessons, and Prices.
* **Pricing Table**: A clear breakdown of lesson costs.
* **Hero Section**: A bold headline over the logo.

## Tech Used:
* HTML5 & CSS3
* Git & GitHub for Version Control
